# Language Translator 🌐

A sleek, modern, and lightweight web-based language translator. It offers instant, client-side translation across dozens of global languages, styled with a premium dark mode and a fully responsive layout.

---

## ✨ Features

* **⚡ Instant Translation**: Integrates with the Google Translate API for real-time text translation as you type.
* **🌓 Dual Theme support**: Smooth toggle between Light and Dark modes with custom sun and moon icon transitions.
* **🔄 Language Swap**: Quickly swap the source and target languages (including their text) with a single click.
* **🌐 Auto-Detect**: Automatically detects the input language.
* **📊 Character Counter**: Shows character limit tracking (up to 5,000 characters) dynamically.
* **📱 Fully Responsive**: Fluid design optimized for desktops, tablets, and mobile devices.

---

## 🛠️ Built With

* **HTML5**: Structured semantic layout.
* **Vanilla CSS3**: Tailored styling using variables for seamless light/dark mode support and custom transition animations.
* **JavaScript (ES6+)**: Custom dropdown selection logic, API integration, and DOM manipulation.
* **Ionicons**: Sleek and modern icons for language selection, dropdowns, and swaps.

## 🚀 How to Run Locally

Since this is a client-side project, you do not need to install any heavy dependencies or run a complex backend.

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Sameerakm1925/Language-Translation.git
   ```
2. **Open the project**:
   * Simply double-click the `index.html` file to open it in your browser.
   * Or, run it using a local development server (such as VS Code's **Live Server** extension).
